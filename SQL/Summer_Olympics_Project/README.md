# Summer Olympics SQL Project

## Data source
This dataset was obtained from DataCamp (www.datacamp.com)

## Dataset overview
Dataset contains 31165 rows and 9 columns
Column attributes and their datatypes are the following:
1) Year INTEGER
2) City VARCHAR
3) Sport VARCHAR
4) Discipline VARCHAR
5) Athlete VARCHAR
6) Country VARCHAR
7) Gender VARCHAR
8) Event VARCHAR
9) Medal VARCHAR
